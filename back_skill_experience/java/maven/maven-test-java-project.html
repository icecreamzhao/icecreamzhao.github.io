  • 运行测试用例

直接执行mvn test, 会直接运行该项目下的所有测试用例。
如果想要运行TestPracticeOne.java下的testOne()方法, 可以这样: mvn -Dtest=TestPracticeOne#testOne test, 就可以了, 这种方式也支持正则, 比如mvn -Dtest=TestPracticeOne#test* test, 这样运行的是所有开头是test的方法。
