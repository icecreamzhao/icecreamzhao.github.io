摘要

首先总结一下本篇博客的内容: 将一个全新的树莓派通过执行脚本的方式将我所有的使用习惯和开发环境下载到本地并编译, 该脚本通过命令选项的方式将所需要的可选项设置好, 这样脚本在执行中途就不需要等待用户输入了。

最后将所有的软件安装位置和安装的成功与否生成一个txt文本文档保存在用户根目录下。

本脚本如果在安装某个软件失败了, 需要自行解决并安装下一个软件。
